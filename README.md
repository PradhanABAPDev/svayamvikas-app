## Quick Start (5 commands to first build)

```bash
# 1. Clone and enter project
git clone https://github.com/YOUR_USERNAME/svayamvikas-app
cd svayamvikas-app

# 2. Copy your game into www/
cp path/to/your/game.html www/index.html
cp path/to/your/game.js   www/game.js

# 3. Run the one-command setup
node scripts/setup.js

# 4. Open Android Studio
npm run android:open

# 5. Run on device/emulator
npm run android:run
```

## Step-by-Step: Play Store

### 1. Set up your environment
```bash
# Install Android Studio (free): https://developer.android.com/studio
# Install Java 17: https://adoptium.net
node scripts/setup.js
```

### 2. Configure your Worker URL
Replace `YOUR-WORKER.workers.dev` in:
- `www/index.html` (Content-Security-Policy connect-src)
- `android-patches/network_security_config.xml`
- `ios-patches/Info.plist` (NSExceptionDomains)

### 3. Deploy Cloudflare Worker (MUST do before submitting)
```bash
# Install Wrangler
npm install -g wrangler
wrangler login

# Set your API key as a secret (NEVER put it in code)
cd cloudflare
wrangler secret put ANTHROPIC_API_KEY
# paste your key when prompted

# Deploy
wrangler deploy worker.ts --name svayamvikas-api
# → Your URL: https://svayamvikas-api.YOUR-SUBDOMAIN.workers.dev
```

### 4. Build release AAB
```bash
npm run android:open
# In Android Studio:
# Build → Generate Signed Bundle/APK → Android App Bundle
# → Create new keystore (SAVE THE PASSWORD!)
# → Release → Finish
```

### 5. Create Play Console account
1. Go to https://play.google.com/console
2. Pay one-time $25 (~₹2,100)
3. Create app → "SvayamVikās"
4. Upload your AAB to Internal Testing first
5. Fill Data Safety questionnaire (AI data: yes, not shared with third parties)
6. Content rating: fill IARC questionnaire (result: Teen/13+)
7. Submit for review → **1–3 days**

## Step-by-Step: App Store

### 1. Register Apple Developer Program
- https://developer.apple.com/programs/enroll/
- $99/year (~₹8,300)

### 2. Build (Mac required, OR use GitHub Actions)

#### Option A: With a Mac
```bash
npm run ios:open
# In Xcode:
# 1. Select your Apple Developer team
# 2. Enable "Automatically manage signing"
# 3. Product → Archive
# 4. Distribute App → App Store Connect → Upload
```

#### Option B: Without a Mac (GitHub Actions)
```bash
# Push your code to GitHub
git add . && git commit -m "Initial release" && git push

# GitHub Actions automatically builds the IPA on Apple Silicon runner
# See: .github/workflows/build.yml → build-ios job
```

### 3. App Store Connect
1. Go to https://appstoreconnect.apple.com
2. My Apps → + → New App
3. Bundle ID: `com.svayam.vikas`
4. Fill metadata (name, subtitle, description, keywords)
5. Upload screenshots (iPhone 6.7" + iPad 12.9" required)
6. Privacy Policy URL: host on GitHub Pages (free)
7. Submit for review → **1–3 days**

## GitHub Secrets Setup

Add these in: GitHub repo → Settings → Secrets → Actions

### Required for Android
| Secret | How to get it |
|--------|--------------|
| `ANDROID_KEYSTORE_BASE64` | `base64 -i release-keystore.jks` |
| `KEYSTORE_PASSWORD` | Password you set during keytool |
| `KEYSTORE_ALIAS_PASSWORD` | Same as above (if you used same password) |

### Required for iOS
| Secret | How to get it |
|--------|--------------|
| `IOS_CERTIFICATE_BASE64` | Export .p12 from Keychain → `base64 -i cert.p12` |
| `IOS_CERTIFICATE_PASSWORD` | Password set when exporting .p12 |
| `IOS_PROVISIONING_PROFILE_BASE64` | Download .mobileprovision → `base64 -i profile.mobileprovision` |
| `APPLE_TEAM_ID` | developer.apple.com → Membership → Team ID |
| `ASC_KEY_ID` | App Store Connect → Users → Keys → Key ID |
| `ASC_ISSUER_ID` | App Store Connect → Users → Keys → Issuer ID |
| `APP_STORE_CONNECT_API_KEY` | Content of the .p8 file downloaded from ASC |

### Required for Cloudflare
| Secret | How to get it |
|--------|--------------|
| `CLOUDFLARE_API_TOKEN` | dash.cloudflare.com → Profile → API Tokens → Create Token |

## Trigger a Release Build

```bash
# Bump version in capacitor.config.ts → versionCode + versionName
# Then tag and push:
git add .
git commit -m "Release v1.0.1"
git tag v1.0.1
git push origin main --tags

# GitHub Actions automatically:
# ✓ Builds Android AAB (signed)
# ✓ Builds iOS IPA (signed)
# ✓ Uploads IPA to TestFlight
# ✓ Deploys updated Cloudflare Worker
# ✓ Creates GitHub Release with AAB attached
```
